Load debug information file (Загрузить файл с отладочной информацией)

 
 Действие    имя: LoadDbgFile
Данная команда позволяет загрузить DBG файл.

Если дизассемблируемая программа дополнительно имеет DBG файл, эта команда может использоваться для загрузки информации из DBG файла в базу данных. IDA автоматически загружает DBG файлы, расположенные в одном каталоге с входным файлом.

TВстроенный загрузчик отладочной информации не поддерживает NB10 и PDB форматы файлов.  he built-in debug information loader cannot load NB10 format files and PDB files. To load those files, please use a special plugin PDB.DLL which can be run manually using Edit->Plugins submenu. This plugin uses MS Windows DLLs to load the debug information and therefore has the following limitations: 

        - it works only under MS Windows
        - it will load only PDBs compatible with the currently
          installed IMAGEHLP.DLL
See also Load... submenu commands.
