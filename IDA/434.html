Load IDS file (Загрузить IDS файл)

 
 Действие    имя: LoadIdsFile
Эта команда загружает IDS файл идентификаторов.

В IDS файле содержится информация о известных функциях (например, как MS Windows API), а именно: 

           

        - их имена

        - их порядковый номер в DLL

        - возможно, содержащий комментарий 

        - количество передаваемых параметров, в стеке 

        - количество сброшенных возвращаемых параметров

Найденные в одноименном каталоге, IDS файлы загружаются автоматически. Данная команда позволяет загружать IDS файл из любого каталога, даже после загрузки в базу данных основного файла.
