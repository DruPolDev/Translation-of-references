File (Меню Файл)
В этом разделе меню вы можете: 
  New instance            Открыть новый экземпляр IDA

  Open                    Открыть новый файл или базу данных

  Load file               Загрузить файл

  Produce output file     Генерировать выходной файл
  Script command          Выполнить команды сценария
  OS shell                Выполнить команды ОС              
  Save database           Сохранить       базу данных в упакованной форме
  Save database as...     Сохранить базу данных в упакованной форме в другом файле
  Take database snapshot  Сделать снимок базы данных
  Abort (Close)           Закрыть IDA без сохранения изменений
  Quit (Exit)             Закрыть IDA с сохранением изменений

Смотрите раздел Главное меню .
