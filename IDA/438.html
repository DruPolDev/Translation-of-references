Produce output files (Создать выходные файлы)
This submenu allows you to produce various output files. It also allows you to unload the database.

The text file-producing operations below will make use of the currently-selected encoding for output files. 

 Generate MAP file
 Generate ASM file
 Generate INC file
 Generate LST file
 Generate EXE file
 Generate DIF file
 Generate HTML file
 Generate flow chart GDL file
 Generate call graph GDL file
 Dump database to IDC file
 Dump typeinfo to IDC file
 Create C header file

See also other File... submenu commands.
