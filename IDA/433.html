Load additional file (Загрузить дополнительный файл)

 
 Действие    имя: LoadFile
Данная команда загружает двоичный файл. Новый файл добавляется в текущую базу данных, и вся информация сохраняется.

TСодержимое файла определяется в программе как неопознаные байты.

С помощью этой команды можно загружать только двоичные файлы.

Смотрите другие команды меню Load file .
