Reload the input file (Перезагрузить входной файл)

 
 Действие    имя: ReloadFile
Данная команда перезагружает уже загруженный входной файл в базу данных. IDA пытается сохранит максимальное количество информации в базе данных. Будут сохранены все имена, комментарии, информация о сегментах и т.п.

Изменяются только числовые значения отдельных байтов.

Эта команда работает только для некоторых типов входных файлов: загрузка в базу данных файлов со специальными настройками, может привести к сбою команды. В этом случае используйте команду Dump database to IDC file  и перезагрузите файл вручную.

Также смотрите другие команды раздела меню Load file .
