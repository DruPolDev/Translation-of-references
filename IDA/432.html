Execute Script File (Выполнить файл сценария)

 
 Действие    имя: Execute
Вы можете выполнить любой файл сценария, поддерживаемый встроенным или добавленным, с помощью плагинов, скриптовым движком (IDC, Python и т.п). Язык сценария определяется исходя из используемого расширения имени файла скрипта.

Также смотрите другие команды меню Load file и Script command.
