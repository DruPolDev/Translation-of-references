How to use the help subsystem
The following keys exist: F1 this screen Shift-F1 help index Ctrl-F1 IDC index Tab next highlighted phrase Shift-Tab prev highlighted phrase Enter follow a link Backspace previous help screen F5 zoom help window Esc close help window 
  arrows        one step movement
  PgUp          Page Up
  PgDn          Page Down
  Ctrl-PgUp     top of the window
  Ctrl-PgDn     bottom of the window
  Home          start of the line
  End           logical end of the line
