Структура каталога установки

Программа установки AutoIt создает структуру каталогов (обычно \Program File\AutoIt3), представленной в следующей таблице. Создаются ярлыки в меню "Пуск", другие файлы не добавляются и не изменяются.

Файлы и каталоги Описание
(Файлы верхнего уровня)
AutoIt3.exe Основная программа AutoIt, единственно необходимый файл для запуска сценариев!
AutoIt3_x64.exe x64-битная версия AutoIt (если установлено).
Au3Info.exe Инструмент AutoIt, для получения информации о графическом оконном интерфейсе.
Au3Info_x64.exe x64-битная версия Au3Info (если установлено).
AU3Check.exe Инструмент проверки синтаксиса AutoIt.
AutoIt.chm Файл справки.
Uninstall.exe Программа удаления AutoIt.
AutoIt v3 Website.url URL ссылка: https://www.autoitscript.com/site/autoit/
Aut2Exe

Icons\ Каталог содержит иконки, используемые для файлов .au3 в Проводнике.

Aut2Exe.exe Компилятор сценариев.

Aut2Exe_x64.exe x64-битная версия Aut2Exe (если установлено).

UPX.exe Инструмент для UPX сжатия файлов.
Examples

COM\ Каталог содержит примеры сценариев AutoIt для COM.

GUI\ Каталог содержит примеры сценариев AutoIt для графических интерфейсов.

Helpfile\ Каталог содержит примеры сценариев AutoIt, используемые в файле справки.
Extras

AutoUpdateIt\ Каталог содержит сценарии для простого обновления версии AutoIt3.

Editors\ Каталог содержит определения подсветки синтаксиса для некоторых популярных редакторов текста.

Geshi\ Каталог содержит файл для подсветки синтаксиса на стороне сервера Geshi .

Prettify\ Каталог содержит JavaScript файл для подсветки синтаксиса на стороне клиента Google Prettify .
Icons

Каталог содержит иконки, используемые для файлов .au3 в Проводнике.
Include

Каталог содержит включаемые файлы (предварительно написанные пользовательские функции). Смотрите документацию по определению пользовательских функций.
AutoItX

Каталог содержит DLL-версию AutoIt3, предоставляющая функционал AutoIt через интерфейс ActiveX/COM и DLL.
SciTE

Содержит облегченную версию редактора SciTE с подсветкой синтаксиса.

Следует отметить, что для запуска сценариев AutoIt требуется только один файл AutoIt3.exe. Если вы компилируете сценарий в исполняемый файл, пользователю не требуется устанавливать AutoIt для запуска скомпилированного исполняемого файла.

Ключи реестра

Установщик AutoIt создает ключи реестра в ветках HKEY_LOCAL_MACHINE\Software\AutoIt v3 или HKEY_LOCAL_MACHINE\SOFTWARE\Wow6432Node\AutoIt v3\ - для 64-битной системы и HKEY_CURRENT_USER\Software\AutoIt v3 . Ключи НЕ создаются/используются без полной установки. AutoIt не устанавливают на серверах и т.д.

В таблице ниже показаны стандартные (или типичные) разделы реестра. Ключи, выделенные курсивом, создаются не самим установщиком, а при первом запуске соответствующей утилиты:
