Редакторы сценариев

Сценарии AutoIt представляют собой простые текстовые файлы, которые можно редактировать в любом текстовом редакторе. Однако существует множество бесплатных или условно-бесплатных редакторов, которые намного лучше подходят для написания сценариев, с возможностью подсветки синтаксиса, так что ключевые слова и функции AutoIt выделяются, что значительно упрощает написание сценариев и снижает вероятность ошибок.

SciTE является стандартным редактором, используемым многими пользователями AutoIt. Команда разработчиков AutoIt разработала модифицированную версию SciTE с подсветкой синтаксиса и интеграцией дополнительных инструментов AutoIt (например, проверкой синтаксиса и форматированием текста). "Lite" версия редактора SciTE поставляется с установочным пакетом AutoIt. Полную версию SciTE4AutoIt3 можно загрузить отдельно по адресу http://www.autoitscript.com/autoit3/scite/. 

 

Другие рекомендуемы редакторы:

  • TextPad
  • Crimson Editor (бесплатно)
  • Source Edit (бесплатно)
  • UltraEdit
  • Notepad++ (бесплатно)

Установочный пакет AutoIt поставляется с определеними подсветки синтаксиса для некоторых популярных редакторов текста, в установочном каталоге Extra (ссылка в меню Старт /AutoIt v3/Extra).
