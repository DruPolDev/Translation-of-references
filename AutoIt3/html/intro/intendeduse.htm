Назначение

AutoIt постоянно развивается как язык программирования. Он создавался как дополнительный инструмент для автоматизации основных задач графического интерфейса других программ (например, получение значения нажатой клавиши или кнопки графического интерфейса), по-прежнему автоматизация процессов лежит в основе AutoIt. Однако с появлением множества новых функций, AutoIt стал более мощным инструментом, чем прежде.

 

Вот лишь некоторые из новых и обновленных функций:

  • Автоматизация графического интерфейса, создание графического интерфейса своего приложения.
  • COM функциональнось (Объектная модель), заполняющая пробелы WSH-языков: VBScript, JScript.
  • Циклы, функции и регулярные выражения.
  • Огромное количество функций обработки строк.
  • Совместимый с Perl, механизм регулярных выражений, использующий библиотеку PCRE, с родным 16-битным режимом и поддержкой UCP/UTF.
  • Мощная функция преобразования рекурсивного списка файлов в массив.
  • Простой вызов функций Win32 API и внешних DLL из ваших сценариев.
