Topic Not Found

The requested topic was not found. The following are possible causes and solutions.

  • Cause: The topic has a bad link.

    Solution: Send feedback about the topic. To provide feedback for a topic on the MSDN site, go to the page and click "Click to Rate and Give Feedback". To provide feedback for an offline topic, go to the page and click "Contact Us".

  • Cause: You are using a local installation of the MSDN Library, but you did not install the component that contains the topic.

    Solution: Install the complete SDK documentation using the latest MSDN CD.

  • Cause: You are using a local installation of the SDK, but you did not install the component that contains the topic.

    Solution: Install the complete SDK documentation from the latest SDK.


Note that you can also download the latest SDK documentation from the Microsoft Download Center or view the SDK online on MSDN.
