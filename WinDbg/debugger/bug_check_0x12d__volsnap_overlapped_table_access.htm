Bug Check 0x12D: VOLSNAP_OVERLAPPED_TABLE_ACCESS

The VOLSNAP_OVERLAPPED_TABLE_ACCESS bug check has a value of 0x0000012D. This indicates that a volsnap tried to access a common table from two different threads which may result in table corruption and eventually corrupt the table.

Important This topic is for programmers. If you are a customer who has received a blue screen error code while using your computer, see Troubleshoot blue screen errors.

VOLSNAP_OVERLAPPED_TABLE_ACCESS Parameters
