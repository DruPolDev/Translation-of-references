Bug Check 0x13C: INVALID_IO_BOOST_STATE

The INVALID_IO_BOOST_STATE bug check has a value of 0x0000013C. This indicates that a thread exited with an invalid I/O boost state. This should be zero when a thread exits.

Important This topic is for programmers. If you are a customer who has received a blue screen error code while using your computer, see Troubleshoot blue screen errors.

INVALID_IO_BOOST_STATE Parameters

The following parameters are displayed on the blue screen.

Parameter Description
1 Pointer to the thread which had the invalid boost state
2 Current boost state
3 Reserved
4 Reserved
