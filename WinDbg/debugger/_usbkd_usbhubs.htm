!usbkd.usbhubs

The !usbkd.usbhubs command displays information about USB hubs.

!usbkd.usbhubs a[v]
!usbkd.usbhubs x[v]
!usbkd.usbhubs r[v]

Parameters

a

Display all hubs.

r

Display root hubs.

x

Display external hubs.

v

The output is verbose. For example, !usbhubs rv displays verbose output about all root hubs.

DLL

Usbkd.dll
