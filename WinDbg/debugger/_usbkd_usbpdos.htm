!usbkd.usbpdos

The !usbkd.usbpdos command displays information about all physical device objects (PDOs) created by the USB hub driver.

!usbkd.usbpdos

DLL

Usbkd.dll

Examples

Here's an example of the output of the !usbpdos command.
