Enable heap free checking

The Enable heap free checking flag validates each heap allocation when it is freed.

Abbreviation

hfc

Hexadecimal value

0x20

Symbolic Name

FLG_HEAP_ENABLE_FREE_CHECK

Destination

System-wide registry entry, kernel flag, image file registry entry
