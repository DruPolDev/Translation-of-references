Bug Check 0x151: UNSUPPORTED_INSTRUCTION_MODE

The UNSUPPORTED_INSTRUCTION_MODE bug check has a value of 0x00000151. This indicates that an attempt was made to execute code using an unsupported processor instruction mode (for example, executing classic ARM instructions instead of ThumbV2 instructions). This is not permitted.

Important This topic is for programmers. If you are a customer who has received a blue screen error code while using your computer, see Troubleshoot blue screen errors.

UNSUPPORTED_INSTRUCTION_MODE Parameters

The following parameters are displayed on the blue screen.

Parameter Description
1 Program counter when the problem was detected.
2 Trap Frame
3 Reserved
4 Reserved
