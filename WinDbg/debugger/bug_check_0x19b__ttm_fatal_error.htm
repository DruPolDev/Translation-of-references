Bug Check 0x19B: TTM_FATAL_ERROR

The TTM_FATAL_ERROR bug check has a value of 0x0000019B. This indicates that the terminal topology manager experienced a fatal error.

Important This topic is for programmers. If you are a customer who has received a blue screen error code while using your computer, see Troubleshoot blue screen errors.

TTM_FATAL_ERROR Parameters

The following parameters are displayed on the blue screen.

Parameter Description
1

Failure type

0x1 : An terminal object could not be generated.

2 - The NT status code of the failure
3 - Reserved
4 - Reserved
2 See parameter 1

4 See parameter 1
