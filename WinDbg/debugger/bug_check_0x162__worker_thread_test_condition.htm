Bug Check 0x163: WORKER_THREAD_TEST_CONDITION

The WORKER_THREAD_TEST_CONDITION bug check has a value of 0x00000163. This indicates that a test for kernel worker threads raised a failure.

Important This topic is for programmers. If you are a customer who has received a blue screen error code while using your computer, see Troubleshoot blue screen errors.

WORKER_THREAD_TEST_CONDITION Parameters

The following parameters are displayed on the blue screen.

Parameter Description
1 Active test flags
2 Flag corresponding to the test that triggered the failure
3 Reserved
4 Reserved
