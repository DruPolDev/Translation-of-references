!popolicy

The !popolicy extension displays the power policy of the target computer.

!popolicy [Address]

Parameters

Address

Specifies the address of the power policy structure to display. If this is omitted, then nt!PopPolicy is displayed.

DLL

Windows 2000

Kdextx86.dll

Windows XP and later

Kdexts.dll

 

Additional Information

To view the system's power capabilities, use the !pocaps extension command. For information about power capabilities and power policy, see the Windows Driver Kit (WDK) documentation and Microsoft Windows Internals, by Mark Russinovich and David Solomon. (These resources may not be available in some languages and countries.)
