!bthkd.sdpnode

The !bthkd.sdpnode command displays information about a node in an sdp tree.

!bthkd.sdpnode addr [flags]

Parameters

addr

Address of the sdp tree node to display.

flags

0x1 - Recurse node

0x2 - Verbose

default is 0x0

DLL

Bthkd.dll
