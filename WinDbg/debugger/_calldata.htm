!calldata

The !calldata extension displays performance information in the form of procedure call statistics from the named table.

    !calldata Table

Parameters

Table

Name of the table that collects the call data.

DLL

Windows 2000

Kdextx86.dll

Windows XP and later

Kdexts.dll
