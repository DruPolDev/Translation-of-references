.echotime (Show Current Time)

The .echotime command displays the current time.

    .echotime

Environment

Modes

User mode, kernel mode

Targets

Live, crash dump

Platforms

All

 

Remarks

The .echotime command displays the time to the computer that the debugger is running on.
