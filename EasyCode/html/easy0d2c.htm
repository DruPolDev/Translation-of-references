SetEnabled method

Available for all objects.

Prototype

SetEnabled Proto hWnd:HWND, bEnabled:BOOL

Syntax

Invoke SetEnabled, hWnd, bEnabled

Function

Sets the Enabled property for an object.

Parameters

hWnd

Handle to the object.

bEnabled

A BOOL value specifying the new enable state for the object.

Return value

Eax/Rax returns TRUE if successful, or FALSE if not.
