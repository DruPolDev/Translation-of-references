Is64BitVersion method

Prototype

Is64BitVersion Proto

Syntax

Invoke Is64BitVersion

Function

Determines whether the running operating system is a 64-bit version or not.

Parameters

None

Return value

Eax/Rax specifies whether the running operating system is a 64-bit version.


Possible returned values are:

TRUE    The running operating system is a 64-bit version.
FALSE  The running operating system is not a 64-bit version (32-bit version).
