GetVisibleRows method

Available for Combo and ImageCombo objects.

Prototype

GetVisibleRows Proto hWnd:HWND

Syntax

Invoke GetVisibleRows, hWnd

Function

Gets the VisibleRows property for an object.

Parameters

hWnd

Handle to the object.

Return value

Eax/Rax returns the current number of visible rows that will be displayed when the combo is dropped.
