The application icon (all project type)

The icon associated with an application is that shown by Windows with the file name inside folders or on the desktop. In Easy Code applications this icon is the first file in the Image Table having an extension of ico, and in visual projects it is also the one shown in the Project Propeties. You can change the order of images by right-clicking files on the Image Table and using the context menu. If there is no icon in the project, Easy Code sets the default Windows icon for applications.

IMPORTANT: If you want to use a cursor file (cur extension) as the aplication icon, change the extension of the cur file to ico and put it as the first icon file in the Image Table.
