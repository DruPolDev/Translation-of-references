GetType method

Available for all objects

Prototype

GetType Proto hWnd:HWND

Syntax

Invoke GetType, hWnd

Function

Gets the Type property for an object.

Parameters

hWnd

Handle to the object.

Return value

Eax/Rax returns the type for the object.
