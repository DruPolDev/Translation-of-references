SetValue method

Available for Check, Radio, HScroll, VScroll, ProgressBar, Slider and UpDown objects.

Prototype

SetValue Proto hWnd:HWND, lValue:DWord (for 32-bit projects)

SetValue Proto hWnd:HWND, lValue:QWord (for 64-bit projects)

Syntax

Invoke SetValue, hWnd, lValue

Function

Sets the Value property for an object.

Parameters

hWnd

Handle to the object.

lValue

New value for the object.

Return value

For Check, Radio, HScroll, VScroll and Slider objects, Eax/Rax returns TRUE if successful, or FALSE if not.
For ProgressBar and UpDown objects, Eax/Rax returns the previous value.


REMARKS: For Check and Radio objects lValue must be BST_CHECKED or BST_UNCHECKED, while for HScroll, VScroll, ProgressBar, Slider and UpDown objects, lValue must be a value between MinValue and MaxValue properties.
