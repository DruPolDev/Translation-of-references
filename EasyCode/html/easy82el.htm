LTrim method

Prototype

LTrim Proto lpszString:LPSTR

Syntax

Invoke LTrim, lpszString

Function

Trims all space and tab characters from the left.

Parameters

lpszString

Pointer to the effective address of the null-terminated string to be left-trimmed.

Return value

Eax/Rax = The length, in characters, of the left-trimmed string.


REMARKS: The LTrim method expects lpszString to be a Unicode string if the application is running as Unicode, or an ANSI string if not (see the IsAppUnicode method). You can call LTrimA to trim ANSI strings in Unicode applications, or LTrimW to trim Unicode strings in ANSI applications.
