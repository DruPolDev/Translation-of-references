Object properties (only visual project type)

In visual projects you have objects. Each object has properties which can modify its appearance and/or behavior. In the right part of the Easy Code interface there is the properties window. That window allows you to modify the values for properties of each object at design time. In most cases, properties can be modified at run time too (see Getting and setting properties).
