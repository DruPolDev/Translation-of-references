SetCueBanner method

Available for Edit and Combo objects.

Prototype

SetCueBanner Proto hWnd:HWND, lMode:DWord, lpszTextW:LPWSTR

Syntax

Invoke SetCueBanner, hWnd, lMode, lpszTextW

Function

Sets the textual cue, or tip, that is displayed by the edit control to prompt the user for information.

Parameters

hWnd

Handle to the object whose textual cue is to be set.

lMode

A value specifying whether the cue banner should show even when the edit control has focus.

lpszTextW

A pointer to a Unicode string that contains the text to display as the textual cue.

Possible values for lMode are:

TRUE    The cue banner should show even when the edit control has focus.
FALSE  The cue banner disappears when the user clicks in the control.

REMARKS: The textual cue is that text in gray font that is displayed in order to provide information about its function when there is no text in the control. An edit control that is used to begin a search may display Enter search here as a textual cue. You cannot set a cue banner on a multiline edit control or on a rich edit control.
