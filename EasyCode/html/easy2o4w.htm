SetTop method

Available for all objects.

Prototype

SetTop Proto hWnd:HWND, lTop:LONG

Syntax

Invoke SetTop, hWnd, lTop

Function

Sets the Top property for an object.

Parameters

hWnd

Handle to the object.

lTop

A LONG value specifying the new y-position for the object.

Return value

Eax/Rax returns TRUE if successful, or FALSE if not.


REMARKS: The lTop value is intepreted as pixels or twips units depending on the ScaleMode property. For Window, MDIWindow and Picture objects it depends on its own ScaleMode property, while for any other object it depends on the ScaleMode property of its parent (if its parent is a Window, MDIWindow or Picture), or its first ascendant having the ScaleMode property.
