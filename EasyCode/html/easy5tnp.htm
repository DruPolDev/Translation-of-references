GetCursorShape method

Available for all objects.

Prototype

GetCursorShape Proto hWnd:HWND

Syntax

Invoke GetCursorShape, hWnd

Function

Gets the CursorShape property for an object.

Parameters

hWnd

Handle to the object.

Return value

Eax/Rax returns the current cursor shape for the object.

Possible returned values are:

ecDefault 0
ecArrow 1
ecIBeam 2
ecWait 3
ecCross 4
ecUpArrow 5
ecSizeAll 6
ecSizeNWSE 7
ecSizeNESW 8
ecSizeWE 9
ecSizeNS 10
ecNoDrop 11
ecAppStarting 12
ecHelp 13
ecCustom 14
