Custom property

Available for ToolBar, StatusBar, TabStrip, Header, ImageList and Rebar objects.

Read/Write at design time

Allows you to customize the corresponding object. You can add or remove buttons for a ToolBar, panels for a StatusBar, tabs for a TabStrip, images for an ImageList, items for a Header, or bands for a Rebar. Also, you can set text and images. For more information see Customizing objects.
