State property

Only available for Thread objects.

Read/Write at design time

Specifies the state for the thread to be created at run time. It can be one of the following values:

0  Running  The thread will start to immediately run after being created (default)
1  Suspended  The thread will not start to run after being created (it will be suspended)


For more information about thread objects, please see the Thread object topic and the Threads and NetMon examples in the EasyCode.Ms\Examples folder.


REMARKS: In order to change the state for a thread, please refer to the ResumeThread and SuspendThread API functions.
