MultiSelect property

Available for List and ListView object.

Read/Write at design time

Specifies whether a List or ListView objects allow selecting multiple lines.

For List objects, it can be one of the following values:

0  None             Just one line can be selected (default)
1  Simple          Line selection is toggled each time is clicked or double-clicked
2  Extended      Multiple lines can be selected combining the mouse with <Shift> or <Ctrl> keys

For ListView objects, it can be one of the following values:

FALSE   Just one line can be selected (default).
TRUE     Multiple lines can be selected.
