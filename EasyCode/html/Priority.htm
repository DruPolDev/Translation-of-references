Priority property

Only available for Thread objects.

Read/Write at design time

Specifies the priority for the thread to be created at run time. It can be one of the following values:

0  Highest  The thread has a priority 2 points above the priority class
1  Above normal  The thread has a priority 1 point above the priority class
2  Normal  The thread has a normal priority (default)
3  Idle  The thread has the process priority class (see the GetPriorityClass and SetPriorityClass API functions)
4  Below normal  The thread has a priority 1 point below the priority class
5  Lowest  The thread has a priority 2 points below the priority class
